# core

- Internal logic of `msw-dev-tool/react`.
- Manage core logic, type and schema.
- It is based on `zustand` and `msw`.
- Getting ready!

# Install

- npm:

```bash
npm i @msw-dev-tool/core --save-dev
```

- yarn:

```bash
yarn add @msw-dev-tool/core --dev
```

- pnpm:

```bash
pnpm add @msw-dev-tool/core --save-dev
```
